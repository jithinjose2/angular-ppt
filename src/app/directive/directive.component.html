Directives

  • Allows dynamic DOM manipullations
  • There are list of build in directives
  • ngFor
    • Similar for foreach in php
    • Can used for variable or static data
    • Can use array index
  • ngIf
    • When used on an element, the element will be only in DOM if condition is true
    • Accepts value or variable names which resolves to boolean
    • Can use array index
  • ngClass
  • ngSwitch, ngStyle
