Deployment

  • ng buildCommand will create compiled static code
  • Builds can be easily deployed to any server or storage services like S3
  • ng build Will create a build for development, with debugging
  • ng build --prod Will create lightweight production build with less debugging options
  • Production build take care of caching problem with custom file names
  • Notes
    • Angular support url routing using real url segment
    • For this to work, we need to point 404 Error to index.html
