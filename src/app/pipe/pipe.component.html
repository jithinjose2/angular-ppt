Pipes

  • Pipes transform displayed values within a template.
  • It can be used in conjunction with one way binding.
  • It is added using '|' character
  • We can pass parameter to pipe for further customization
  • We can chain multiple pipes, which output of first pipe act as input to the second one
  • Possible to create custom pipes
birthday = new Date(1991, 1, 2);
{{ birthday | date }}
{{ birthday | date:'fullDate' }}
{{ birthday | date:'fullDate' | uppercase }}
