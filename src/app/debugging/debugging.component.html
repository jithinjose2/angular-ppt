Debugging

  • Development build of an application contain resource map.
  • Resource map is a mapping between compiled Js code and TypeScript.
  • Using resource map browser enables us to debug TypeScript code in console
  • Augury chrome plugin make debugging much easier
    • It List component map and router map
    • It shows current of each variables
    • In form it shows the validation statuses
    • Injector graph shows tree structure for Dependency Injection
